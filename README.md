# Telegram-бот для чек-апа давления и образа жизни

Бот для проведения опроса по артериальному давлению и образу жизни с персонализированными рекомендациями.

## Возможности

- ✅ 3 контрольных измерения давления
- ✅ Анализ привычек (сон, питание, стресс)
- ✅ Персонализированный отчет с оценкой рисков
- ✅ Рекомендации по образу жизни
- ✅ Система реферальных ссылок и подарков
- ✅ PHQ-2 и PHQ-9 опросники для оценки психического здоровья

## Установка

### Вариант 1: Запуск через Docker (рекомендуется)

1. Установите [Docker](https://www.docker.com/get-started) и Docker Compose

2. Создайте файл `.env` в корне проекта:
```env
# Токен бота от @BotFather (обязательно)
BOT_TOKEN=your_bot_token_here

# ID администраторов через запятую (опционально)
# Получить свой ID можно у бота @userinfobot
ADMIN_IDS=123456789,987654321
```

   **Важно:** Файл `.env` должен быть в корне проекта, рядом с `docker-compose.yml`

3. Запустите бота:
```bash
docker-compose up -d
```

4. Просмотрите логи:
```bash
docker-compose logs -f
```

Подробнее см. [DOCKER.md](DOCKER.md)

### Вариант 2: Локальный запуск

1. Клонируйте репозиторий или скачайте файлы

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env`:
```env
BOT_TOKEN=your_bot_token_here
ADMIN_IDS=123456789,987654321
```

4. Получите токен бота у [@BotFather](https://t.me/BotFather) в Telegram

5. Запустите бота:
```bash
python main.py
```

## Использование

1. Запустите бота командой `/start`
2. Следуйте инструкциям бота для прохождения опроса
3. В конце получите персонализированный отчет с рекомендациями
4. Поделитесь реферальной ссылкой с друзьями для получения подарка

## Админ-команды

- `/stats` - статистика по опросам и пользователям

## База данных

Бот использует SQLite для хранения данных. База данных создается автоматически при первом запуске.

Таблицы:
- `users` - информация о пользователях и реферальных кодах
- `surveys` - результаты опросов
- `referrals` - реферальные связи и подарки

## Технологии

- Python 3.8+
- aiogram 3.11.0
- aiosqlite для работы с БД
