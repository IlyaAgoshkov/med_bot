# Инструкция по запуску бота

## Быстрый старт

1. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Создайте файл `.env`** в корне проекта:
   ```env
   # Токен бота от @BotFather (обязательно)
   BOT_TOKEN=ваш_токен_бота
   
   # ID администраторов через запятую (опционально)
   # Получить свой ID можно у бота @userinfobot
   ADMIN_IDS=123456789,987654321
   ```
   
   **Важно:** 
   - `BOT_TOKEN` - обязателен, без него бот не запустится
   - `ADMIN_IDS` - опционален, можно оставить пустым или не указывать
   - Файл `.env` должен находиться в корне проекта

3. **Запустите бота:**
   ```bash
   python main.py
   ```

## Получение токена бота

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте полученный токен в файл `.env`

## Получение своего ID в Telegram

1. Найдите бота [@userinfobot](https://t.me/userinfobot) в Telegram
2. Отправьте ему команду `/start`
3. Скопируйте ваш ID (число) в `ADMIN_IDS` в файле `.env`

## Проверка работы

После запуска бота:
1. Откройте Telegram
2. Найдите вашего бота по username
3. Отправьте команду `/start`
4. Начните прохождение опроса

## Структура базы данных

База данных `med_bot.db` создается автоматически при первом запуске и содержит:
- `users` - пользователи и реферальные коды
- `surveys` - результаты опросов
- `referrals` - реферальные связи

## Админ-команды

- `/stats` - статистика по опросам и пользователям
